How it works

Unlike standard Canvas APIs where JavasScript calls fillRect or drawImage thousands of times, this demo uses Shared Memory.

  1. Zig calculates physics for all particles.
  2. Zig writes colors directly to a shared byte array.
  3. JavaScript displays that array as ImageData in one call.

This bypasses the overhead of the JS-to-Native bridge for every single particle.
